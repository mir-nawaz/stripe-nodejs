# NodejsStripePayment
Stripe payment implementation with nodejs server.

Create config file

```
touch .env
```

Paste below configurations

```
STRIPE_SECRET_KEY=sk_test_
STRIPE_PUBLIC_KEY=pk_test_
PORT=3000
```
 
Start Server
 
```
npm run start
```

Fix Linting

```
npm run fix
```
